## Usage

Install the package

```
npm install jian-yang-tslint-formatter --save-dev
```

Then just run tslint form your project directory

```
tslint -p tsconfig.json --formatters-dir ./node-modules/jian-yang-tslint-formatter/dist --format jianyang
```
